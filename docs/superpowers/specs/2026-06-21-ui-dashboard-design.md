# Balance Fit UI 대시보드 설계서

## 목표
- 단순 텍스트만 보이는 초기 화면을 실제 앱처럼 보이는 대시보드 UI로 개선한다.

## 가정
- 사용자 응답 부재로 기본 UX를 채택한다.
- 실시간 센서 연동 전 단계이므로 샘플 상태/조작 가능한 데모 UI를 제공한다.

## 화면 구성
1. 상단 제목/설명
2. 균형 상태 카드(현재 각도, 상태 라벨, 색상)
3. 빠른 조정 버튼(좌/우 기울기 데모)
4. 캘리브레이션 카드(프로필 선택, 실행 버튼)
5. 개선 가이드 카드(현재 상태 기반 안내 문구)

## 상태 규칙
- `abs(angle) <= 1`: 완벽(초록)
- `abs(angle) <= 5`: 약간 기울어짐(노랑)
- 그 외: 심하게 기울어짐(빨강)

## 구현 범위
- `AppRoot`를 카드 기반 Compose 화면으로 변경
- 각도/상태 계산 로직을 함수로 분리해 단위 테스트 추가
- 기존 Android 실행 흐름(MainActivity) 유지

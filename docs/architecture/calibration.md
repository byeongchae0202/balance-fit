# 센서 캘리브레이션 (Calibration)

스마트폰 센서를 정확하게 조정하고 관리하는 전체 캘리브레이션 시스템을 설명합니다.

---

## 캘리브레이션의 목적

스마트폰마다 센서가 조금씩 다릅니다:
- 제조사가 다름 (삼성, 애플, LG 등)
- 출고 시간이 다름
- 센서 부품 미세한 편차

**캘리브레이션은 현재 기기의 센서를 정확하게 조정하는 작업입니다.**

---

## 캘리브레이션 플로우

```
1️⃣ 프로필 선택
   ↓
2️⃣ 센서 상태 체크
   ↓
3️⃣ 캘리브레이션 실행
   ↓
4️⃣ 온도 보정
   ↓
5️⃣ 테스트 모드
   ↓
6️⃣ 정확도 신뢰도 평가
   ↓
7️⃣ 저장 및 히스토리 기록
```

---

## 1단계: 프로필 선택

### 무엇인가?
사용 환경에 따라 다른 캘리브레이션 프로필을 저장합니다.

### 프로필 종류

**실내 프로필 (Indoor)**
```
환경: 실내, 안정적인 온도
예: 사무실, 가정, 실험실
용도: 일반 사용자, 실내 측정
```

**실외 프로필 (Outdoor)**
```
환경: 실외, 변동하는 온도, 햇빛
예: 건설 현장, 야외 작업
용도: 산업용 전문가, 현장 측정
```

### 선택 방법
```
┌──────────────────────┐
│ 캘리브레이션        │
├──────────────────────┤
│ 프로필 선택:        │
│ ◯ 실내 (Indoor)    │
│ ◯ 실외 (Outdoor)   │
│                    │
│ [다음] 버튼         │
└──────────────────────┘
```

---

## 2단계: 센서 상태 체크

### 목적
캘리브레이션 전에 센서가 정상 작동하는지 확인합니다.

### 체크 항목

**✅ 가속도계 점검**
```
화면 표시: "기기를 여러 방향으로 천천히 회전시켜 보세요"

기기를 기울이면:
- 6개 방향 (위, 아래, 좌, 우, 앞, 뒤)에서 센서 값 수신 확인
- 모든 방향에서 값이 변하면 정상 ✓
- 특정 방향에서 반응 없으면 이상 ✗
```

**✅ 자이로스코프 점검**
```
화면 표시: "기기를 천천히 회전해 보세요"

360도 회전하면:
- 회전 감지 확인
- 모든 축(X, Y, Z)에서 값 변화 확인
```

**📊 센서 신뢰도 점수**
```
결과:
- 모든 축 정상: "센서 상태 양호 (100%)"
- 일부 축 미약: "센서 상태 경고 (85%)"
- 심각한 문제: "센서 상태 불량 (40%) - 기기 수리 필요"
```

---

## 3단계: 캘리브레이션 실행

### 전 준비

사용자에게 지시:
```
┌──────────────────────┐
│ 준비 단계            │
├──────────────────────┤
│ 1. 기기를 평탄한    │
│    표면에 놓으세요   │
│ 2. 움직이지 않게    │
│    고정하세요        │
│ 3. [시작] 버튼      │
└──────────────────────┘
```

### 자동 캘리브레이션 (3초)

```
화면 표시:
┌──────────────────────┐
│ 캘리브레이션 중...  │
│                    │
│  3초 후 시작...     │
│  [████░░░░] 30%    │
│                    │
│ 기기를 움직이지     │
│ 마세요!             │
└──────────────────────┘

내부 동작:
1초 대기 (사용자가 기기를 정착시킬 시간)
↓
현재 센서 값 100개 수집
↓
평균값 계산 (노이즈 제거)
↓
기준값으로 저장
↓
"✅ 캘리브레이션 완료"
```

### 저장되는 기준값
```
{
  profile: "indoor",              // 프로필 타입
  timestamp: "2026-06-13 10:50",  // 캘리브레이션 시간
  baselineValues: {
    x: 0.0,
    y: 0.0,
    z: 9.8
  },
  temperature: 22.5,              // 캘리브레이션 당시 온도
  calibrationQuality: 98%         // 정확도
}
```

## 4단계: 온도 보정

### 온도가 센서에 영향을 주나?

**YES!** 온도 변화는 센서 값에 영향을 줍니다:
```
실내 (22°C):
센서 기준값 = Z: 9.8

실외 (5°C):
센서 값이 약 0.5% 달라질 수 있음
```

### 온도 보정 방법

**기기의 온도 센서 사용:**
```
캘리브레이션 실행 시점의 기기 온도 자동 측정
↓
향후 측정할 때 현재 온도 측정
↓
온도 차이에 따라 자동으로 센서 값 조정
```

### 예시
```
캘리브레이션:
- 시간: 오후 2시
- 온도: 22°C
- 기준값: Z = 9.8

1시간 후 측정:
- 현재 온도: 20°C (2도 낮음)
- 센서 보정값: Z = 9.75 (자동 조정됨)
- 조정된 각도 계산에 사용
```

## 5단계: 테스트 모드

### 목적
캘리브레이션이 제대로 작동하는지 사용자가 직접 확인합니다.

### 테스트 화면
```
┌──────────────────────┐
│ 테스트 모드          │
├──────────────────────┤
│ 기기를 천천히      │
│ 기울여 보세요      │
│                    │
│  균형 시각화:      │
│     ╱  ┐          │
│    ╱   │ 5°       │
│   ╱    └          │
│                    │
│ "정상 작동 중"     │
└──────────────────────┘
```

### 동작
1. 사용자가 기기를 천천히 기울임
2. 화면의 원이 기울임에 따라 실시간으로 회전
3. 각도 값이 실시간으로 변함
4. 사용자가 "정상 작동" 또는 "이상 발생" 선택

---

## 6단계: 정확도 신뢰도 평가

### 신뢰도 점수 계산

캘리브레이션 후 정확도를 평가합니다:

```
평가 항목:
┌─────────────────────────────────┐
│ 1. 센서 안정성         : 95%    │
│    (센서 노이즈 수준)           │
│                                 │
│ 2. 환경 안정성         : 92%    │
│    (캘리브레이션 중 움직임)    │
│                                 │
│ 3. 온도 일관성         : 88%    │
│    (온도 변동 수준)             │
│                                 │
│ 4. 테스트 모드 검증    : 96%    │
│    (사용자 확인)                │
│                                 │
│ ─────────────────────────────    │
│ 최종 정확도            : 92%    │
└─────────────────────────────────┘
```

### 정확도별 해석

```
95% 이상    : 🟢 "매우 우수 - 산업용으로도 신뢰할 수 있음"
90~94%      : 🟢 "우수 - 일반 사용에 무리 없음"
85~89%      : 🟡 "양호 - 다시 캘리브레이션 권장"
80% 미만    : 🔴 "주의 - 캘리브레이션을 다시 해주세요"
```

## 빠른 재캘리브레이션

### 상황
새로운 장소에서 빠르게 캘리브레이션해야 할 때 (산업용 전문가).

### 방법

**표준 캘리브레이션**: 3초 대기 + 테스트 모드 = 약 30초

**빠른 재캘리브레이션**: 1초 대기만 = 약 5초

```
┌──────────────────────┐
│ 빠른 재캘리브레이션  │
├──────────────────────┤
│ 기존 프로필:         │
│ 실내 (92% 신뢰도)   │
│                    │
│ 테스트 건너뛰고    │
│ 빠르게 적용하시겠어요?
│                    │
│ [예] [아니오]       │
└──────────────────────┘
```

### 동작
```
기존 설정 값: X=0, Y=0, Z=9.8 (기준)
↓
새 환경에서 1초 센서 데이터 수집
↓
약간의 오프셋만 조정
↓
"✅ 빠른 캘리브레이션 완료"
↓
미세 조정된 새 기준값 사용
```

## 자동 재캘리브레이션 제안

### 언제 제안할까?

**조건 1: 시간 기반**
```
마지막 캘리브레이션으로부터 7일 경과
↓
알림: "캘리브레이션을 다시 해보세요"
```

**조건 2: 정확도 기반**
```
연속 3회 측정에서 이상한 값 감지
↓
신뢰도가 92% 이하로 떨어짐
↓
알림: "센서 정확도가 낮아졌습니다. 재캘리브레이션을 권합니다"
```

### 알림 화면
```
┌──────────────────────┐
│ 알림                │
├──────────────────────┤
│ 센서 정확도가       │
│ 낮아졌습니다        │
│                    │
│ 현재 신뢰도: 78%   │
│                    │
│ [지금 캘리브레이션] │

```

## 캘리브레이션 상태 아이콘

앱 상단에 항상 표시:

```
🟢 정상     : 신뢰도 90% 이상, 최근 7일 이내 캘리브레이션
🟡 주의     : 신뢰도 80~89%, 또는 7일 이상 경과
🔴 필수     : 신뢰도 80% 미만, 또는 센서 이상 감지
```

## 산업용 전문가를 위한 팁

### 다중 기기 관리
```
기기 1: 실내 프로필 (92% 신뢰도)
기기 2: 실외 프로필 (88% 신뢰도)
기기 3: 실내 프로필 (90% 신뢰도)

각 기기의 히스토리 별도 관리
```

### 정확도 기준
```
일반 사용자: 85% 이상 충분
산업 표준: 90% 이상 권장
정밀 측정: 95% 이상 필수
```

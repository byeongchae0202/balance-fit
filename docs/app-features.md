# 앱 기능 설명 (App Features)

Balance Fit 앱의 3가지 핵심 기능을 설명합니다.

---

## 1️⃣ 실시간 균형 상태 표시

### 화면 구성
앱을 실행하면 화면 중앙에 큰 원이 표시됩니다. 이 원은 물체의 균형 상태를 실시간으로 보여줍니다.

### 동작 방식
- 🔘 **원의 기울기**: 기기(스마트폰)가 기울어진 각도만큼 원이 기울어집니다
  - 예: 왼쪽으로 5도 기울어지면 원도 왼쪽으로 5도 기울어짐
  - 예: 오른쪽으로 3도 기울어지면 원도 오른쪽으로 3도 기울어짐

- 📊 **각도 숫자 표시**: 원 안에 현재 각도를 숫자로 표시
  - 예: "+5°" (오른쪽으로 5도), "-3°" (왼쪽으로 3도)

- 🎨 **색상 피드백**: 균형 상태에 따라 원의 색상이 변함
  - 🟢 **초록색**: 완벽하게 균형잡힘 (±1° 이내)
  - 🟡 **노란색**: 약간 기울어짐 (±2~5°)
  - 🔴 **빨간색**: 심하게 기울어짐 (5° 이상)

- ⚡ **실시간 업데이트**: 센서 데이터가 계속 들어오므로 원이 실시간으로 변함

---

## 2️⃣ 센서 캘리브레이션

### 목적
스마트폰 기기마다 센서의 특성이 조금씩 다릅니다. 캘리브레이션은 현재 기기의 센서를 정확하게 조정하는 작업입니다.

### 사용 방법

**단계 1: 캘리브레이션 시작**
- 앱 메뉴에서 "설정" → "센서 캘리브레이션" 버튼 클릭

**단계 2: 기기 준비**
- 화면에 "기기를 평탄한 표면에 놓으세요" 라는 지시문이 나타남
- 책상이나 바닥 같은 완전히 평평한 곳에 스마트폰을 내려놓음

**단계 3: 자동 캘리브레이션**
- 3초 카운트다운 시작 (3... 2... 1...)
- 카운트다운이 끝나면 자동으로 현재 센서 값을 기준점(0도)으로 설정

**단계 4: 완료**
- "✅ 캘리브레이션 완료!" 메시지 표시
- "확인" 버튼을 눌러 빠져나옴

### 언제 캘리브레이션을 해야 할까?
- ✅ 앱을 처음 설치했을 때
- ✅ 측정 값이 정확하지 않을 때
- ✅ 다른 기기에서 앱을 사용할 때

## 3️⃣ 균형 개선 가이드

### 목적
현재 물체가 기울어져 있을 때, 어떻게 조정하면 균형을 맞출 수 있는지 알려줍니다.

### 화면 표시
"실시간 균형 상태 표시" 아래에 텍스트로 조언이 나타납니다.

### 예시
균형이 잘못되어 있을 때:

```
🔴 균형 상태: 오른쪽으로 4도 기울어짐

→ 물체를 왼쪽으로 약 2cm 옮기세요
```

물체를 조정하면:

```
🟡 균형 상태: 오른쪽으로 1도 기울어짐

→ 물체를 왼쪽으로 약 0.5cm 추가로 옮기세요
```

완벽하게 균형잡히면:

```
🟢 균형 상태: 완벽함!

✅ 물체가 균형잡혀 있습니다
```

### 동작 방식
- 📍 **실시간 업데이트**: 센서가 계속 감지하므로 사용자가 물체를 조정할 때마다 지시문이 갱신됨
- 📐 **구체적 수치 제시**: 단순히 "왼쪽으로 옮기세요"가 아니라 "왼쪽으로 약 2cm"처럼 구체적인 거리 제시
- 🎯 **단계별 안내**: 크게 기울어져 있으면 큰 이동 제시, 거의 균형잡혀 가면 세밀한 조정 제시

---

## 기능 연동

이 3가지 기능은 함께 동작합니다:

1. **캘리브레이션** 실행 → 센서 기준 설정
2. 앱을 실행하면 **균형 상태 표시**로 현재 상태를 시각화
3. 물체가 기울어져 있으면 **개선 가이드**로 조정 방법 제시
4. 사용자가 물체를 조정하면 **균형 상태 표시**와 **개선 가이드**가 실시간 업데이트

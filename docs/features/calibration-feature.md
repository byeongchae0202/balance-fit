# 캘리브레이션 기능 (Calibration Feature)

센서 기준점을 맞춰 측정 정확도를 확보하는 기능입니다.

---

## 목적

- 기기별 센서 편차를 보정합니다.
- 실내/실외 환경별 프로필을 분리해 재사용성을 높입니다.
- 자동 재캘리브레이션으로 장기 정확도를 유지합니다.

---

## 사용자 플로우

### 1) 진입
- `설정 > 센서 캘리브레이션`

### 2) 프로필 선택
- `실내 프로필` / `실외 프로필`
- 이미 저장된 프로필이 있으면 마지막 사용값을 기본 선택

### 3) 준비 안내
- 평탄한 표면에 기기 배치 안내
- 흔들림 최소화 문구 표시

### 4) 측정
- 3초 카운트다운 후 기준값 저장
- 필요 시 2~3회 샘플 평균값 사용

### 5) 검증
- 짧은 검증 단계(예: 1초)로 오차 범위 확인
- 성공 시 저장, 실패 시 재시도 안내

---

## UI 구성

- 상단: 현재 선택 프로필(실내/실외)
- 본문:
  - 준비 상태 카드
  - 카운트다운 표시
  - 진행 상태(수집 중/검증 중/완료)
- 하단 버튼:
  - `시작`
  - `다시 시도`
  - `저장 후 종료`

---

## 상태 모델(권장)

- `IDLE`: 대기
- `GUIDE`: 준비 안내
- `COUNTDOWN`: 카운트다운
- `SAMPLING`: 센서 샘플 수집
- `VALIDATING`: 검증
- `SUCCESS`: 완료
- `FAILED`: 실패/재시도 필요

상태 전환은 UI와 1:1로 매핑해 디버깅이 쉽도록 유지합니다.

---

## 자동 재캘리브레이션 정책

- 기본 주기: 7일
- 조기 트리거:
  - 센서 신뢰도 지속 저하
  - 측정 변동성이 임계값 초과
- 자동 실행 전 사용자에게 안내(즉시/나중에 선택)

---

## 예외/오류 처리

### 표면 불안정 감지
- 메시지: `기기가 흔들리고 있어 정확한 캘리브레이션이 어렵습니다.`
- 해결: 안정화 후 재시도 버튼

### 샘플 수 부족
- 원인: 앱 백그라운드 전환, 센서 이벤트 누락
- 처리: 수집 단계로 자동 복귀, 횟수 제한 초과 시 실패 처리

### 저장 실패
- 로컬 저장 실패 시 명확한 오류 표시
- 저장 실패 상태에서 완료 표시 금지

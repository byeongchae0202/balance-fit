# 균형 개선 가이드 기능 (Improvement Guide)

현재 기울기 상태를 바탕으로 사용자가 취해야 할 조정 행동을 안내하는 기능입니다.

---

## 목적

- "현재 상태"를 "어떻게 고칠지"로 변환합니다.
- 사용자가 한 번에 이해할 수 있는 짧고 구체적인 행동 문구를 제공합니다.
- 실시간 피드백으로 반복 조정을 돕습니다.

---

## 핵심 출력

### 상태 요약
- 예시: `오른쪽으로 4.0° 기울어짐`

### 행동 제안
- 예시: `물체를 왼쪽으로 약 2cm 옮기세요`

### 진행 피드백
- 예시: `거의 맞았습니다. 왼쪽으로 0.5cm 추가 이동하세요`

---

## 안내 문구 생성 규칙

1. 기울기 방향 반대로 이동 지시
2. 각도 크기에 따라 이동량 단계화
3. 문구 길이는 한 줄 중심으로 유지

권장 단계 예시:

| 절대 각도 | 안내 강도 | 예시 안내 |
|---|---|---|
| 0°~1° | 완료 | `균형이 잘 맞습니다.` |
| 1°~3° | 미세 조정 | `왼쪽으로 약 0.5~1cm 이동하세요.` |
| 3°~7° | 일반 조정 | `왼쪽으로 약 1~3cm 이동하세요.` |
| 7° 이상 | 큰 조정 | `왼쪽으로 3cm 이상 이동 후 다시 측정하세요.` |

---

## 화면 구성

- 상단: 현재 균형 상태(색상/텍스트)
- 중단: 추천 행동 카드(핵심 문장 1개)
- 하단: 보조 팁
  - `천천히 이동하면서 값을 확인하세요`
  - `먼저 큰 조정 후 미세 조정을 진행하세요`

---

## 상호작용 흐름

1. 센서 업데이트 수신
2. 최신 각도/방향 계산
3. 안내 문구 생성
4. UI 갱신
5. 사용자가 물체 이동
6. 상태 변화에 따라 문구 재생성

완벽 상태가 일정 시간 유지되면:
- 성공 메시지 + 진동/사운드(선택) 제공

---

## 예외/오류 처리

### 값 흔들림이 심한 경우
- 문구를 즉시 바꾸지 않고 짧은 안정화 구간(예: 300ms) 적용

### 방향 불명확(0 근처 진동)
- `현재 거의 균형입니다. 미세 조정만 진행하세요.` 표시

### 캘리브레이션 누락 의심
- `정확도 향상을 위해 캘리브레이션을 권장합니다.` CTA 노출

---

## 구현 체크리스트

- [ ] 각도→문구 매핑 테이블 정의
- [ ] 문구 생성기(방향/강도/단위) 구현
- [ ] 실시간 갱신 안정화(과도한 깜빡임 방지) 적용
- [ ] 완료 상태 유지 감지 로직 추가
- [ ] 캘리브레이션 권장 조건 연동

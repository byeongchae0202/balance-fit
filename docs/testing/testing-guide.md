# 테스트 가이드 (Testing Guide)

Balance Fit 앱의 기능 정확도와 안정성을 검증하기 위한 테스트 기준 문서입니다.

---

## 테스트 목표

- 센서 입력부터 UI 출력까지 핵심 흐름이 기대대로 동작하는지 확인
- 캘리브레이션/개선 가이드의 정확도와 일관성 검증
- 기기/환경 차이에 따른 오차를 허용 범위 내로 관리

---

## 테스트 환경

### 필수 구성
- Android Studio 최신 안정 버전
- 실제 안드로이드 기기 1대 이상
- 평탄한 표면(기준 측정용)

### 권장 구성
- 제조사/센서 칩셋이 다른 기기 2~3종
- 실내/실외 각 1개 환경

---

## 테스트 유형

## 1) 기능 테스트

### A. 균형 상태 표시

1. 앱 실행 후 중앙 상태 원/각도/색상 노출 확인
2. 기기를 좌/우로 각각 기울였을 때 방향/수치 일치 확인
3. 임계값 경계(-1°, +1°, -5°, +5° 근처)에서 상태 전환 정확성 확인

기대 결과:
- 값/색상/라벨이 동시에 일관되게 변경되어야 함

### B. 캘리브레이션

1. 실내/실외 프로필 선택 후 캘리브레이션 실행
2. 카운트다운 → 샘플링 → 검증 → 저장 순서 확인
3. 저장 후 앱 재실행 시 프로필 값 재사용 확인

기대 결과:
- 성공 시 명확한 완료 상태 표시
- 실패 시 원인 메시지와 재시도 동선 제공

### C. 균형 개선 가이드

1. 기울기 방향 반대로 이동 안내가 나오는지 확인
2. 각도 변화에 따라 이동 권장량 단계가 바뀌는지 확인
3. 완벽 상태 유지 시 성공 메시지 표시 확인

기대 결과:
- 문구가 구체적이며 방향/강도가 물리 상태와 일치해야 함

---

## 2) 정확도 테스트

### 기준 측정 테스트
- 완전히 평탄한 표면에서 10회 측정
- 허용 기준(권장): 평균 절대 오차 1.0° 이하

### 반복성 테스트
- 동일 조건에서 30초 동안 측정값 분산 확인
- 허용 기준(권장): 급격한 스파이크 빈도 최소화

### 프로필 비교 테스트
- 실내/실외 프로필 각각 동일 물체 측정
- 프로필 전환 시 의도한 보정 차이가 반영되는지 확인

---

## 3) 회귀 테스트

아래 변경 시 반드시 재실행:
- 임계값(constants) 수정
- 필터/계산 로직 수정
- 캘리브레이션 상태 전환 로직 수정
- 가이드 문구 생성 규칙 수정

회귀 체크 최소 항목:
- 상태 전환 정확성
- 캘리브레이션 저장/로드
- 가이드 문구 방향/단계

---

## 4) 장치/환경 테스트

### 장치별
- 저가형/중급형/고성능 기기에서 UI 프레임 저하 여부 확인
- 센서 미지원/품질 저하 기기에서 오류 메시지 확인

### 환경별
- 실내(평탄/진동 적음) vs 실외(진동/바람 가능) 비교
- 환경 노이즈가 높은 경우 신뢰도 표시/재캘리브레이션 권장 동작 확인
